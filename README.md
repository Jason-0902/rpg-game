# 無盡首領試煉：玩家手冊

這不是技術文件，這是一份給玩家看的「遊戲資料總覽」。

## 遊戲目標
- 持續挑戰更高樓層的敵人。
- 每 50 層遇到更強的 Boss。
- 500 層後進入深淵模式。
- 1000 層後進入天界模式。
- 若與惡魔簽約，後續敵人將改為人類討伐隊。

## 職業與定位
- 戰士：高血量高防禦，前排穩定。
- 法師：高攻擊高爆發，防禦較低。
- 刺客：高爆擊、高連段收割。

## 職業進階（每 50 級）
- 50 級：第一階進化技能
- 100 級：第二階進化技能
- 150 級：第三階進化技能
- 進階後保留舊技能，並新增進階技能。

## 樓層怪物分佈
- 1~50 層（新手村）：
  - 綠史萊姆
  - 洞窟哥布林
  - 荒野幼狼
  - 暗穴蝙蝠
  - 非 Boss 怪物攻擊與血量上限不超過 50
- 51~500 層（地表）：
  - 熔爪惡狼
  - 墓地怨骸
  - 雷鳴守衛
  - 夜蝕祭司
  - 隨樓層提高混入更強怪池
- 501~1000 層（深淵）：
  - 深淵噬魂者
  - 虛空監視者
  - 猩紅判官
- 1001+ 層（天界）：
  - 天穹審判者
  - 聖曜羽騎
  - 星環主教
- 惡魔契約後（人類討伐線）：
  - 羅因聖堂騎士
  - 薇爾聖劍使
  - 希爾維亞聖女
  - 艾德里克勇者

## 事件系統機率
系統流程是「先判定商店，再判定事件」。

### 商店機率
- 每次戰鬥勝利後，商店出現機率：9%

### 事件觸發率（未進商店時）
- 1~50 層：36%
- 51+ 層：34%

### 事件內部權重（已觸發事件後）
- 惡魔事件：0.7%（僅人類陣營可觸發）
- 神祇/星座事件：0.6%
- 1001+ 層天界事件：28%
- 501+ 層深淵事件：24%
- 一般負面事件：16%
- 其餘為一般正面事件

### 一般事件池
- 正面：
  - 遺跡金庫
  - 古老卷軸
  - 守護祝福
  - 幸運補給箱
- 負面（皆為暫時效果，不是永久扣屬）：
  - 毒霧陷阱
  - 路匪勒索
  - 詛咒石碑

### 稀有事件池
- 神祇/星座：
  - 曜星女神・露米娜菈
  - 銀弦天琴座・艾莉希昂
- 惡魔：
  - 魔君「莫札薩斯」
  - 紅月魔女「瓦爾卡菈」
  - 深獄公爵「奈瑟里昂」

## 惡魔契約機制
- 遭遇惡魔後，可能簽約或脫離。
- 簽約時：
  - 大幅提升能力值
  - 獲得對應職業的魔契技能
  - 陣營改為 `demon`
  - 後續敵方改為人類陣營
  - UI 主題改為暗紅色

## 裝備系統

### 裝備部位（8 種）
- 頭盔
- 手套
- 武器
- 盾牌
- 項鍊
- 鞋子
- 護甲
- 腿甲

### 稀有度（5 階）
- 一般
- 精良
- 高等
- 傳說
- 神話

### 武器名稱池（依稀有度）
- 一般：鐵風直刃、獵狼短刀、赤紋戰槍、霜鋼手斧、荒原重劍
- 精良：蒼雷裂牙、影鴉穿雲、寒星斷岳、風嵐斬夜、白焰逐日
- 高等：黑曜天誓、星塵審判、燼羽龍嘯、渦雷魔槍、月魄天裁
- 傳說：滅界神鋒、寂滅蒼穹、深淵王權、曜炎天命、幽煌聖戟
- 神話：終焉天啟、虛空神臨、無限審判者、星海主宰之刃、永夜創世槍

### 裝備效果方向
- 武器：主加攻擊
- 盾牌：主加防禦
- 頭/衣服/腿：主加最大生命
- 手套/項鍊：攻擊 + 爆擊
- 鞋子：防禦 + 爆擊

## 藥水系統
- 初級藥水：回復最大生命 25%
- 中級藥水：回復最大生命 50%
- 高級藥水：回復最大生命 75%
- 頂級藥水：直接回滿
- 本作生命回復只能透過藥水，不可自然回復。

## 技能取得來源
- 基礎技能：開局職業自帶
- 掉落技能：戰鬥/事件中機率取得
- 進階技能：每 50 級職業進階取得
- 神祇/星座技能：超低機率事件取得
- 惡魔契約技能：惡魔簽約後取得

## 戰鬥成長規則
- 每次擊敗敵人：一定獲得金錢
- 每次擊敗敵人：隨機成長 1~4 種屬性
  - 攻擊 +1~20
  - 防禦 +5~50
  - 爆擊 +1~10%
  - 最大生命 +10~100
- 爆擊率超過 100% 時，溢出值會轉成爆擊傷害額外乘區。
